About This Project

PolicyGrid is an engineering-level simulation that models distributed AI governance, ethical firewalls, multi-actor authority transitions, and healing mechanisms for misused AI clusters.

This project combines a web-based visualization and a Python-driven simulation engine to show how a controlled AI ecosystem can regain balance.
